Un valor de 0.5 suele ser adecuado para documentos con espacio suficiente. 2. filename: El nombre del archivo PDF generado. 3. image: • type: Define el formato de imagen (por ejemplo, 'jpeg' o 'png'). • quality: La calidad de la imagen (valor entre 0 y 1, donde 0.98 es alta calidad). 4. html2canvas: • scale: Controla la escala de captura de la página. Un valor de 2 duplica la resolución y mejora la calidad del PDF. Puedes usar valores mayores (2.5, 3), pero ten en cuenta que incrementará el tamaño del archivo y el tiempo de generación. 5. jsPDF: • unit: Unidad de medida ('in' para pulgadas, 'mm' para milímetros, 'pt' para puntos). • format: Tamaño del papel ('letter', 'A4', etc.). • orientation: 'portrait' (vertical) o 'landscape' (horizontal).
